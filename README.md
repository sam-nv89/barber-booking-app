# 💈 Barber Booking App

Современное веб-приложение для онлайн-записи к барберам/мастерам. Работает как PWA и Telegram Mini App.

## ✨ Возможности

### Для клиентов
- 📅 Онлайн-запись на услуги
- 📱 "Мой QR" для быстрой регистрации визита
- 🕐 Выбор удобного времени
- 📱 Просмотр истории визитов
- ⭐ Оценка и отзывы о визитах
- 🔔 Уведомления о статусе записи

### Для мастеров
- 📊 Аналитика и статистика (выручка, клиенты, загруженность)
- 💱 Мультивалютность (KZT, RUB, USD, EUR, TRY и др.)
- 📷 QR-сканер для проверки посетителей
- 📋 Управление записями (подтверждение, отмена, перенос)
- ⚙️ Настройка услуг и расписания
- 👥 База клиентов с историей
- 🛡️ Блокировка нежелательных клиентов
- 💬 Ответы на отзывы

## 🛠️ Технологии

- **React 18** + **Vite** — быстрая сборка и HMR
- **Tailwind CSS** — стилизация
- **Zustand** — управление состоянием
- **Framer Motion** — анимации
- **date-fns** — работа с датами
- **Lucide React** — иконки
- **QR Code** — генерация и сканирование

## 🌐 Локализация

Приложение поддерживает 5 языков:
- 🇷🇺 Русский
- 🇰🇿 Қазақша
- 🇬🇧 English
- 🇪🇸 Español
- 🇹🇷 Türkçe

## 🚀 Запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build
```

## 📱 Telegram Mini App

Приложение оптимизировано для работы как Telegram Mini App:
- Адаптивный интерфейс
- Интеграция с Telegram WebApp API
- Поддержка темной/светлой темы

