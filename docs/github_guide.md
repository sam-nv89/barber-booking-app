# Инструкция по работе с GitHub

Эта инструкция поможет вам сохранить ваш проект в облаке GitHub и развернуть его на другом компьютере.

> [!TIP]
> **Автоматическая отправка**: Я могу автоматически отправлять изменения. Просто напишите мне **"/push_to_github"** или **"Залей изменения"**.

## 1. Подготовка (Первый раз)

### Установка Git
Если Git не установлен, скачайте его с [git-scm.com](https://git-scm.com/) и установите.

### Создание репозитория на GitHub
1. Зайдите на [github.com](https://github.com) и войдите в аккаунт.
2. Нажмите **New Repository** (Новый репозиторий).
3. Введите название (например, `barber-booking-app`).
4. Выберите **Private** (если хотите скрыть код) или **Public**.
5. Нажмите **Create repository**.

## 2. Загрузка проекта в облако (С вашего текущего ПК)

Откройте терминал (PowerShell или CMD) в папке проекта `d:\App Development\Antigravity` и выполните команды по очереди:

```bash
# 1. Инициализация Git (если еще не делали)
git init

# 2. Добавление всех файлов
git add .

# 3. Сохранение изменений (коммит)
git commit -m "Initial commit"

# 4. Привязка к GitHub (замените URL на ваш!)
# Ссылку можно взять на странице созданного репозитория
git remote add origin https://github.com/ВАШ_НИК/barber-booking-app.git

# 5. Отправка кода на GitHub
git branch -M main
git push -u origin main
```

## 3. Развертывание на другом компьютере

1. Установите **Node.js** и **Git** на новом компьютере.
2. Откройте терминал в папке, где хотите создать проект.
3. Скачайте проект (клонирование):
   ```bash
   git clone https://github.com/ВАШ_НИК/barber-booking-app.git
   ```
4. Зайдите в папку:
   ```bash
   cd barber-booking-app
   ```
5. Установите зависимости:
   ```bash
   npm install
   ```
6. Запустите проект:
   ```bash
   npm run dev
   ```

## 4. Повседневная работа

### Сохранить изменения (на рабочем ПК)
```bash
git add .
git commit -m "Описание изменений (например: добавил скидки)"
git push
```

### Получить изменения (на другом ПК)
```bash
git pull
```
